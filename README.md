## Getting Started

### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone this project

```sh
git clone https://github.com/bwangsta/movie-night.git
```

2. Install required dependencies

```sh
npm install
```

3. Run

```sh
npm run dev
```
